# IT-Infrastructure-Labs

This repository showcases hands-on labs and projects completed through my Business Information Technology diploma and personal home lab experimentation. The work demonstrates practical experience building, configuring, and troubleshooting enterprise-style IT environments.

## Overview
These labs simulate real-world administrative and support scenarios involving system deployment, identity management, networking configuration, and automation. The focus is on applying foundational infrastructure concepts rather than theoretical exercises.

## Areas Covered

### Active Directory & Identity Management
- Domain controller deployment
- Organizational Unit design
- User and group administration
- Permissions and access control

### Windows Server Administration
- Server configuration and management
- File and print services
- Endpoint integration
- System validation and testing

### Networking Fundamentals
- TCP/IP configuration and troubleshooting
- DNS and DHCP setup
- IP addressing and connectivity validation
- Virtual network testing

### Virtualization
- Multi-VM infrastructure setup
- Server and client deployment
- Environment simulation using Hyper-V / VMware

### Automation
- PowerShell scripting
- CSV-based user provisioning
- Administrative task automation

### Cloud Exposure
- Azure virtual machine deployment
- Basic cloud networking exploration

## Technologies Used
- Windows Server
- Active Directory
- VMware / Hyper-V
- PowerShell
- Microsoft Azure
- TCP/IP Networking

## Purpose
This repository serves as a technical portfolio demonstrating applied infrastructure skills, problem-solving workflows, and continuous learning within IT systems support and administration domains.
